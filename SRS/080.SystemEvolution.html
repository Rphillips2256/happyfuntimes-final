System Evolution

Evolution for Game of Life: • Game will allow online play, allowing players from all regions to play each other • Game will have a large database to keep track of user data • Game will give people more challenging life decisions • Game will make people pay different bill such as hospital, dental, etc. • Game will allow more then one user to have an income • Game boards will change so players don’t always have the same game every time • Game can allow users to communicate with each other during play, to make game more competitive • Game will have a better GUI as Graphics improve to make graphics more interesting • Game will allow more players, instead of up to six it will allow up to 12 • Game will allow different colored spaces, with different options
